# Character Counter App

This is a React application that allows users to type text and see real-time statistics.

## Features
- Character count
- Word count
- Estimated reading time

## How it works
The app uses React useState to track user input and updates the statistics as the user types.

## Components
- TextInput: Handles user input
- StatsDisplay: Displays the calculated statistics

## How to run
npm install
npm run dev
